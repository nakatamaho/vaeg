# Milestone task archive policy

Task documents are executable instructions only when the maintainer assigns
that exact milestone and the prerequisite gate in `../ROADMAP.md` has passed.
Once its gate passes, a task document becomes a historical record and must not
be started again merely because it remains under this directory.

In particular, M36 through M41 are completed Z80-migration records (G36–G41
passed). Their checked-in task files are archive-only. The active uPD9002
sequence is likewise governed by the gate statuses in the roadmap; at the M51
implementation SHA, M42 through M50 are archive-only and M51 is the sole
current human-gated task.

Do not infer authorization for an old draft, an earlier sequence, or a later
milestone from its presence here. Follow the single task explicitly assigned
by the maintainer.
